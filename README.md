# Trust the Tuesday

> Everyone blamed pricing for churn. I found the whole trial was decided by what a user did on their very first Tuesday.

The question: Could a single early action predict whether a trial user would ever pay?
The result: A first week action predicted churn far ahead, with strong model AUC.

## Run
    pip install -r requirements.txt
    python main.py

## Stack
Python, SQL, Looker (this demo uses scikit-learn gradient boosting).
